# WickhamsPlugin
## 插件功能

 1. **登录系统** 使用join命令注册及登录。（**支持正版从你我做起**）
 2. 自动保存地图系统，可配置的保存时间间隔。
 3. 自动公告系统，简陋的自动定时循环公告。
 4. 加入游戏时的自定义通知，简陋的一句话加入通知。~~例：这个服务器吊炸天~~
 5. 保护耕地系统，防止**人为**踩坏耕地。
 6. 死亡是否保留背包内的物品开关，**统一设置，所有世界都生效。**
 7. OP指令：tp、tpall、gm。
 8. 玩家指令：tpa、tpayes、spawn、back、home、sethome。
 9. 可选血量随着等级而增加。
